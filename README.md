# Analytics SaaS Platform

A white-label analytics dashboard SaaS platform where marketing agencies can provide clients with professional Google Analytics reports.

## Tech Stack

- **Next.js 14** with App Router
- **TypeScript**
- **Tailwind CSS**
- **Supabase** (Database, Auth, Storage)
- **Google Analytics Data API v1**

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables (see `.env.local.example`)

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app` - Next.js App Router pages and layouts
- `/components` - React components
- `/lib` - Utility functions and configurations
- `/types` - TypeScript type definitions
- `/hooks` - Custom React hooks
